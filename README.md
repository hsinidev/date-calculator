## ğŸŒŸ Introduction

**Doodax** is a high-precision, modern web application designed to solve complex date arithmetic problems. While simple subtraction can give you milliseconds, Doodax provides a human-centric breakdown of time, calculating years, months, weeks, and days with algorithmic accuracy that accounts for leap years and irregular month lengths. Wrapped in a stunning, animated cosmic interface, it offers both utility and aesthetics.

## âœ¨ Key Features

- **Algorithmic Precision**: Custom-built date logic in `src/utils/DateMath.ts` ensures 100% accuracy for edge cases like leap years and month-end rollovers.
- **Deep Analytics**: Breakdown of total duration into Days, Weeks, and Hours alongside the standard Year/Month/Day format.
- **Immersive UX**: A multi-layered, animated nebula background provides a calming, modern user experience.
- **SEO Optimized**: Built with Server-Side Rendering principles in mind, including extensive JSON-LD schema, semantic HTML5, and a comprehensive 3,500+ word knowledge base.
- **Privacy First**: All calculations happen client-side. No data is ever sent to a server.
